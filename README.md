# BankAccountManager
This Java application simulates a bank account management system. The project focuses on implementing core banking functionalities through Object-Oriented Programming (OOP) principles, ensuring data integrity and logical consistency during transactions.

Project Description
This Java application simulates a bank account management system. The project focuses on implementing core banking functionalities through Object-Oriented Programming (OOP) principles, ensuring data integrity and logical consistency during transactions.

Technical Features
Balance Management: Implementation of secure methods for depositing and withdrawing funds.

Transaction Validation: Logic to prevent invalid operations, such as negative inputs or overdrafts without sufficient funds.

Encapsulation: Use of private modifiers to protect account data and public methods for controlled access.

Console Interface: Interactive command-line interface for user input handling and real-time balance updates.
